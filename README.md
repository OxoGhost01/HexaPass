# HexaPass

⚠️ Pre-alpha / Experimental

HexaPass is an experimental Android password manager built for learning and research purposes.

DO NOT use this application to store real or sensitive credentials.
Security guarantees are not complete yet.

## Status
- Core vault open / read / edit / delete: in progress
- Autofill service: not implemented
- Integrated SuperCool keyboard: not implemented
- Audit tools: not implemented
- Security review: not done
